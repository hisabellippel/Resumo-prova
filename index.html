HTML

O que é uma tag? Uma tag é um marcador de texto. Temos várias tags:
A tag div, a sua função é separar os conteúdos do código HTML em
divisões de espaço/conjunto, sem qualquer peso semântico.


A tag script, sua função é criar conexões com arquivos.


O footer, ele é o rodapé da página, e é uma tag semântica


O H1 até H6, definine os títulos e os tamanhos.


O p é para fazer parágrafo. O main é onde fica o conteúdo principal da página.


Header é o cabeçalho. Body é o corpo do código, tudo que vai ficar visível.


A tag label é umtexto justificativo do Input, nela você usa a tag form também


A tag img, que insere imagens. Section, divide o site em seções.


Button, botão, tem duas formas de se usar, type"_" e oneclick
